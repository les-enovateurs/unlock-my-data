# Importer les favoris et données de Google Chrome vers Firefox

Résumé fidèle du guide officiel Mozilla : https://support.mozilla.org/fr/kb/importer-favoris-chrome

## Démarrer l’importation

Vous pouvez importer vos données Chrome dans Firefox de deux façons :

### Depuis les paramètres de Firefox
1. Ouvrez Firefox.
2. Dans la barre des menus, cliquez sur **Firefox** puis **Paramètres**.
3. Dans le panneau **Général**, section **Importer des données d’un navigateur**, cliquez sur **Importer des données**.
4. L’assistant d’importation s’ouvre. Sélectionnez le profil Chrome et les données à importer.

### Depuis la fenêtre de la bibliothèque
1. Cliquez sur le menu **Marque-pages** puis sur **Afficher tous les marque-pages**.
2. Dans la fenêtre Bibliothèque, cliquez sur **Importer et sauvegarder** > **Importer des données d’un autre navigateur…**
3. Suivez l’assistant d’importation.

## Choisir les données à importer

Dans l’assistant, sélectionnez le profil Chrome et les données à importer :
- Mots de passe enregistrés
- Marque-pages
- Historique de navigation
- Extensions compatibles
- Données de formulaires et moyens de paiement

Cliquez sur **Importer** puis sur **Terminé**.

## Importer les mots de passe Chrome via CSV
1. Exportez vos mots de passe Chrome au format CSV (via le gestionnaire de mots de passe Chrome).
2. Dans l’assistant d’importation Firefox, choisissez **Mots de passe depuis un fichier CSV**.

## Finaliser l’installation des extensions importées
Après l’importation, Firefox propose de terminer l’installation des extensions compatibles importées de Chrome.

## Afficher la barre personnelle (pour retrouver vos marque-pages)
1. Cliquez sur le menu **Outils supplémentaires** > **Personnaliser la barre d’outils…**
2. En bas, menu **Barres d’outils** > **Barre personnelle** (choisissez l’affichage souhaité).
