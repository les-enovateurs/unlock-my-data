## Start the import

You can import your Chrome data into Vivaldi in two ways:

### On first launch of Vivaldi
1. Download and install Vivaldi: https://vivaldi.com/download/
2. On first launch, the import wizard opens automatically.
3. Select "Google Chrome" as the source browser.
4. Choose which items to import (bookmarks, passwords, history, search engines, etc.).
5. Click "Import".

### From the Vivaldi menu
1. Vivaldi menu (top left) → **File** → **Import from another browser…**
2. Select "Google Chrome" and follow the wizard.

## Import from a file
You can also import bookmarks, passwords, or history from an HTML or CSV file exported from Chrome.

## Key points
- Some Chrome passwords or extensions may not be imported automatically.
- Imported bookmarks appear in the "Imported" folder on the Vivaldi bookmarks bar.
