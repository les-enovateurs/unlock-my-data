
## Start the import process

You can import your Chrome data into Firefox in two ways:

### From Firefox Settings
1. Open Firefox.
2. In the menu bar, click **Firefox** and select **Settings** (or Preferences).
3. In the **General** panel, under **Import browser data**, click **Import Data**.
4. The Import Wizard opens. Select the Chrome profile and the data you want to import.

### From the Library window
1. Click the **Bookmarks** menu, then **Manage bookmarks**.
2. In the Library window, click **Import and Backup** > **Import Data from Another Browser…**
3. Follow the Import Wizard.

## Choose data and complete the import

In the Import Wizard, select the Chrome profile and the data to import:
- Saved passwords
- Bookmarks
- Browsing history
- Compatible extensions
- Form autofill data and payment methods

Click **Import** and then **Done**.

## Import Chrome passwords via CSV
1. Export your Chrome passwords as a CSV file (via Chrome Password Manager).
2. In the Firefox Import Wizard, choose **Passwords from CSV file**.

## Complete imported extensions installation
After import, Firefox will prompt you to finalize installation of compatible extensions imported from Chrome.

## Show the Bookmarks Toolbar (to find your imported bookmarks)
1. Click the **More Tools…** menu > **Customize Toolbar…**
2. At the bottom, use the **Toolbars** dropdown > **Bookmarks Toolbar** (choose your preferred display).
