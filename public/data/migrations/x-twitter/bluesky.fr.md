## Étape 1 : Archiver vos données X/Twitter
Pour quitter X/Twitter, vous pouvez vouloir garder une copie de vos données avant de partir. Cela inclut vos tweets, abonnés et abonnements.

**Comment demander votre archive :**
- Rendez-vous sur [Demander votre archive à X/Twitter](https://www.escape-x.org/Leaving-Twitter-X-Step-1-Request-Your-Archive-from-X-Twitter.html)
- Ou directement : [Paramètres d’archive X/Twitter](https://x.com/settings/download_your_data)
- Aide officielle : [Accéder à vos données X](https://help.x.com/fr/managing-your-account/accessing-your-x-data)

X/Twitter demandera une authentification forte (mot de passe, SMS/email). Après la demande, attendez 1 à 3 jours pour votre archive. Téléchargez le fichier .ZIP quand il est prêt.

## Étape 2 : Créer un compte sur Bluesky
- Visitez [https://bsky.app](https://bsky.app) ou téléchargez l’application Bluesky pour Android/iPhone.
- Voir le guide : [Créer un compte sur Mastodon et/ou Bluesky](https://www.escape-x.org/Leaving-Twitter-X-Step-2-I-Create-an-Account-on-Mastodon-and-or-BlueSky.html)

## Étape 3 : Reconnecter avec votre communauté
Utilisez des plateformes comme OpenPortability pour retrouver vos contacts :
- Rendez-vous sur [OpenPortability](https://openportability.org/fr/auth/signin)
- Téléversez votre archive pour retrouver et vous reconnecter avec vos abonnés et abonnements.

Tout se fait localement dans votre navigateur. Aucun mot de passe n’est stocké, seulement les données nécessaires à la reconnexion. Toutes les données partagées sont détruites après le processus.
