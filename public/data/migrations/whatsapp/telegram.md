# Migrer vers Telegram

Telegram est une alternative populaire avec beaucoup de fonctionnalités.

1. **Installer Telegram**.
2. **Importer vos historiques de discussion WhatsApp** (Fonctionnalité officielle) :
   - Ouvrez une discussion WhatsApp.
   - Appuyez sur le nom du contact > "Exporter la discussion".
   - Choisissez "Sans médias" ou "Avec médias".
   - Dans le menu de partage, sélectionnez Telegram.
   - Choisissez la discussion correspondante dans Telegram pour importer l'historique.
