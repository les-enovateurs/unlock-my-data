# Migrer vers Signal

Signal est reconnu comme la référence en matière de messagerie privée.

1. **Installer Signal** : Disponible sur iOS et Android.
2. **Inviter vos contacts** :
   - Signal vous indiquera quels contacts utilisent déjà l'application.
   - Envoyez un lien d'invitation aux autres.
3. **Créer un groupe de migration** :
   - Sur WhatsApp, créez un groupe "On bouge sur Signal !" avec le lien de votre nouveau groupe Signal.
