# Migrer de Notion vers Slite

## Comment exporter une page unique depuis Notion
1. Ouvrez la page dans Notion.
2. Cliquez sur le menu … en haut à droite et choisissez **Exporter**.
3. Utilisez les paramètres suivants :
	- Format d’exportation : **HTML**
	- Inclure les bases de données : **Vue par défaut**
	- Inclure le contenu : **Tout**
	- Inclure les sous-pages : **Activé**
	- Créer des dossiers pour les sous-pages : **Activé**
	- Exporter les commentaires : **Désactivé**
4. Cliquez sur **Exporter** et attendez le téléchargement de l’archive.

## Comment exporter un espace de travail Notion
1. Cliquez sur le nom de votre espace de travail en haut à gauche et choisissez **Paramètres**.
2. Allez dans **Espace de travail > Général**.
3. Dans la section Exporter, cliquez sur **Exporter** à côté du contenu de l’espace de travail.
4. Utilisez les paramètres suivants :
	- Format d’exportation : **HTML**
	- Inclure les bases de données : **Vue par défaut**
	- Inclure le contenu : **Tout**
	- Inclure les sous-pages : **Activé**
	- Créer des dossiers pour les sous-pages : **Activé**
	- Exporter les commentaires : **Désactivé**
5. Cliquez sur **Exporter** et attendez le téléchargement de l’archive.

## Comment importer votre contenu dans Slite
1. Cliquez sur **Importer** en bas de la barre latérale gauche.
2. Sélectionnez l’option **Notion**.
3. Choisissez **Importer un ou plusieurs documents** ou **Importer tout l’espace de travail Notion** et suivez les instructions à l’écran.
4. Choisissez un nom pour le nouveau canal dans lequel vos documents seront importés.
5. Choisissez la visibilité de votre nouveau canal :
	- **Ouvert** : tout le monde dans votre espace de travail peut accéder à ce canal.
	- **Privé** : seules les personnes que vous choisissez peuvent accéder à ce canal.
6. Une fois l’import réussi, une confirmation s’affichera en bas à droite. Cliquez sur **Aller au document** pour voir vos documents importés.
7. Si une erreur indique que le canal n’a pas pu être créé, contactez l’administrateur de votre espace de travail pour obtenir les droits nécessaires.

En savoir plus : [Guide d’import Slite](https://slite.slite.page/p/KYFLSp6oBKT0jW/Importing-from-Notion)

## Ce qui est importé
- La structure des documents (ce que vous voyez dans la barre latérale de Notion)
- Le contenu des documents, y compris :
  - Titres, listes à puces, listes numérotées, etc.
  - Blocs d’astuces
  - Liens entre documents
  - Images
  - Vidéos
- Les bases de données seront converties en Collections dans Slite

En savoir plus : [Ce qui est importé](https://slite.slite.page/p/8bd4HVVH-4KoaC)
