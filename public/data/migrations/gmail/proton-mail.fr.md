## Étapes pour activer le transfert automatique Gmail → Proton Mail
1. Ouvrez Proton Mail sur le web : `Paramètres` → `Tous les paramètres` → `Importer via Easy Switch`.
2. Cliquez sur « Configurer le transfert automatique depuis Gmail ».
3. Connectez-vous à votre compte Google/Gmail et autorisez Proton à accéder à votre boîte Gmail.
4. Confirmez les autorisations demandées.
5. Proton Mail vous confirmera que le transfert est activé.

## Désactiver le transfert automatique
1. Retournez dans `Importer via Easy Switch` dans Proton Mail.
2. Dans la section « Historique », cliquez sur « Supprimer » à côté du transfert à désactiver.

## Points importants
- Les messages transférés sont chiffrés à l’arrivée sur Proton Mail.
- Les spams ne sont généralement pas transférés.
- Supprimer ou archiver un message dans Proton Mail n’affecte pas Gmail.
- Vous pouvez filtrer ou organiser les e-mails transférés dans Proton Mail.
