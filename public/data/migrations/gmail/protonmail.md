# Migrer vers Proton Mail

1. **Créer un compte Proton Mail** : [proton.me/mail](https://proton.me/mail).
2. **Importer vos emails et contacts** :
   - Utilisez l'outil **Easy Switch** de Proton (dans les paramètres).
   - Connectez votre compte Google pour importer automatiquement vos anciens emails.
3. **Configurer la redirection** :
   - Dans Gmail, activez le transfert automatique vers votre nouvelle adresse Proton pour ne rien manquer pendant la transition.
