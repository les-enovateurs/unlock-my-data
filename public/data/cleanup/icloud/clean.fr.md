1. **Faire le tri dans iCloud Drive**
   - Sur votre iPhone/iPad : Allez dans **Fichiers** > **Explorer** > **iCloud Drive**.
   - Ou sur PC/Mac : Connectez-vous sur [icloud.com](https://www.icloud.com/iclouddrive).
   - Triez par taille pour repérer les documents volumineux inutiles et supprimez-les. N'oubliez pas de vider le dossier "Suppressions récentes".

2. **Nettoyer la photothèque iCloud (Photos)**
   - Ouvrez l'application **Photos**.
   - Allez dans **Albums** > **Types de média** (Vidéos, Captures d'écran, Doublons).
   - Les vidéos prennent le plus d'espace, commencez par là.
   - Videz obligatoirement l'album **Supprimés récemment** pour récupérer le stockage immédiatement.

3. **Gérer les Sauvegardes iPhone/iPad**
   - Sur votre iPhone : **Réglages** > [Votre Nom] > **iCloud** > **Gérer le stockage du compte** > **Sauvegardes**.
   - Supprimez les sauvegardes d'anciens appareils que vous ne possédez plus.
   - Désactivez la sauvegarde dans le cloud pour les applications qui n'en ont pas besoin.
